## Currency Exchange API (Node.js)

A simple API built with Node.js and Express that fetches real-time exchange rates using an external API.

### Features

- Fetches USD, EUR, and BTC exchange rates in BRL

- Uses Axios to consume an external API

- Returns data in JSON format

- Simple and fast API structure

- Technologies Used

- Node.js

- Express

- Axios

- External API (AwesomeAPI)
